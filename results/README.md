# Results

Run outputs are written here:

```text
results/<config_id>/<run_id>/
```

Example:

```text
results/01-baseline/001-20260626-173041-33e1187b/
```

Each run directory should contain:

- `config.yaml`
- `metrics.json`
- `manifest.json`
- `predictions.jsonl`

Keep raw results here. Regenerated figures belong in `figures/`.
